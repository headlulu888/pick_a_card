Это программа, которая выбирает произвольную карту из колоды.

Сойдайте рабочую папку и перейдите в нее С помощью терминала, затем
склонируйте программу командой ниже.

Что бы скачать файл напишите команду:

git clone https://github.com/headlulu888/pick_a_card.git

Перейдите в папку с программой:

cd pick_a_card и запустите файл командой

ruby pick_a_card.rb
